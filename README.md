\# Shopping Cart Simulation



A Java Web Application built using JSP, Servlets, JDBC and PostgreSQL.



\## Technologies Used

\- Java

\- JSP

\- Servlet

\- JDBC

\- PostgreSQL

\- Apache Tomcat

\- HTML / CSS



\## Features

\- Add Product

\- View Products

\- Update Product

\- Delete Product

\- Database Integration



\## Project Architecture

JSP (UI)

↓

Servlet

↓

DAO

↓

JDBC

↓

PostgreSQL Database
